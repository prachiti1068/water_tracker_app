<h1 align="center">Water Tracker App</h1>

<p align="center">
A Flutter application that helps users track their daily water intake and maintain hydration goals.
</p>

<h2>Features</h2>

<ul>
  <li>Set a daily water intake goal</li>
  <li>Add water intake (+250ml, +500ml, +1000ml)</li>
  <li>Track progress with a progress indicator</li>
  <li>View daily summary</li>
  <li>Reset daily intake</li>
  <li>Store data locally using SharedPreferences</li>
</ul>

<h2>Technologies Used</h2>

<ul>
  <li>Flutter</li>
  <li>Dart</li>
  <li>SharedPreferences</li>
  <li>Git</li>
  <li>GitHub</li>
</ul>
